#+TITLE: GUI — TODO

Execution checklist for the GUI module. =readme.org= is the specification;
this file is what remains to do and the resume point for the next agent.

Status keys: TODO (not started), WIP (in progress), DONE.
Order within "First slice" is the intended build order.

* First slice — win32, hello-world-gui without hacks  [COMPLETE]

Built and verified: =./nob build app hello-world-gui win32= succeeds and the
app runs on a Windows host (window titled "Hello World Main", reactor-driven,
no stubs, no nob.c change needed — the existing win32 platform chain picks up
=modules/gui/src= and =src/win32=).
